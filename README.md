## Installation
To install the necessary dependencies, run the following command:

```bash
npm install
npm install express
```

## Usage
To start the application, use the following commands:

```bash
# Build TypeScript
npm run build

# Start the server
npm start
```

Visit `http://localhost:3000` to see the MartialDev platform in action!
